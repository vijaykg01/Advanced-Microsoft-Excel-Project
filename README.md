# Excel Data Analysis Portfolio | Dashboard, Functions & Data Tools

## 📊 Project Overview

This repository showcases a comprehensive set of Excel-based data analysis projects covering core and advanced functionalities used in real-world business scenarios. It includes hands-on implementations of data cleaning, transformation, and visualization techniques using tools such as Pivot Tables, dashboards, lookup functions, conditional formatting, and data validation.

## 📁 Files Included & What They Show

### 1. Dashboard with Pivot Tables (P T with DASHBOARDS.xlsm)

* Interactive dashboard creation
* Sales/data trend analysis
* KPI tracking using charts and slicers
* Business insight generation

### 2. Functions & Formulas

* Use of core Excel formulas
* Logical functions, text functions, and calculations
* Data transformation and manipulation

### 3. VLOOKUP, INDEX & MATCH

* Data lookup and retrieval techniques
* Comparison between traditional and advanced lookup methods
* Efficient data mapping across tables

### 4. Pivot Table Practice

* Data summarization
* Aggregation (SUM, COUNT, AVERAGE)
* Grouping and filtering large datasets

### 5. Sort & Filter (Advanced)

* Multi-level sorting
* Advanced filtering conditions
* Data segmentation

### 6. Conditional Formatting

* Highlighting trends and patterns
* Rule-based formatting
* Visual data interpretation

### 7. Data Validation

* Input control and restrictions
* Dropdown lists
* Error prevention in datasets

### 8. Find, Replace & Paste Special

* Data cleaning techniques
* Bulk editing
* Efficient data handling
